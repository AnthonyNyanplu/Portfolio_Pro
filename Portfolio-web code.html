Analog Signal Conditioning • PCB Design • ADC Interface

ACS712 - Based AC Current Sampling & Signal Conditioning Board for DSP/ADC Interface

This project involved designing and building a current-sensing signal conditioning board using the ACS712 Hall-effect current sensor. The board converts an AC current signal into a DSP-compatible 0 - 3 V voltage signal, filters noise, and prepares the waveform for ADC sampling and software analysis.

Problem

Raw current-sensor output required scaling, offset shifting, and filtering before digital sampling.

Solution

Designed an RC signal-conditioning PCB with voltage scaling and waveform smoothing.

Result

Verified a cleaner ADC-compatible output waveform using oscilloscope measurements.
